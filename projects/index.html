This class project involved creating a Tamagotchi-style game in Processing to practice and apply programming skills. In the game, players care for a virtual pet which has its own storyline. Core gameplay includes feeding and bathing the animal, alongside a minigame with a scoring system. A dynamic time system is also integrated, causing the pet's environment to change and needs such as cleanliness and sleep to arise over time, requiring player attention. This project involved the development of a conceptual webpage exploring the idea of "burning" a song, using "Despacito" as the example, to express the visceral feeling this concept evokes. For its creation, fundamental web technologies were employed: HTML for structure, CSS for visual design and styling, and JavaScript for implementing scripts that brought the interactivity to life. This approach required a deep dive into interactive design concepts, aiming to translate the metaphor of "burning" into a tangible and meaningful digital experience for the user.
